# Remote-Sensing-Change-Detection-On-Land-Using-HRNet
Deep learning-based land change detection using HRNet on bi-temporal satellite images to classify buildings, vegetation, water, and no-change regions
# 🌍 Remote Sensing Change Detection using HRNet

## 📌 Overview
This project focuses on detecting land changes from satellite images taken at different time periods.  
Using deep learning and HRNet architecture, the system identifies and classifies changes such as buildings, vegetation, and water bodies.

---

## 🎯 Problem Statement
Monitoring land changes is crucial for urban planning, environmental analysis, and disaster management.  
Traditional methods are slow and inaccurate. This project aims to automate change detection using deep learning techniques.

---

## 💡 Solution
- Use bi-temporal satellite images (before and after)
- Apply HRNet-based segmentation model
- Detect and classify changes into multiple categories
- Generate pixel-wise change maps

---

## 🧠 Technologies Used
- Python  
- Deep Learning (HRNet)  
- PyTorch / TensorFlow  
- OpenCV  
- NumPy, Pandas  
- Remote Sensing Data  

---

## ⚙️ How It Works
1. Input two satellite images from different time periods  
2. Preprocess and align images  
3. Feed images into HRNet model  
4. Perform semantic segmentation  
5. Generate change map highlighting differences  

---

## 📊 Output Classes
- 🏢 Buildings  
- 🌳 Vegetation  
- 💧 Water  
- ⚪ No Change
